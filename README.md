# 🍀 로또 번호 추천 앱 가이드

행운의 로또 번호 추천기는 1회부터 1212회까지의 **역대 로또 당첨번호 데이터**를 통계학적으로 분석하여, 사용자에게 최적의 번호를 추천하도록 고안된 프리미엄 웹 애플리케이션입니다.

## ✨ 아름다운 프리미엄 디자인
이 앱은 사용자에게 편안하면서도 신비로운 경험을 제공하기 위해 고급스러운 UI를 채택했습니다.
- **글래스모피즘(Glassmorphism)**: 투명하고 은은한 유리 재질감으로 카드 UI를 처리하여 최신 웹디자인 트렌드를 적용했습니다.
- **다이나믹 백그라운드**: 우주의 성운을 떠올리게 하는 고급스러운 색상으로 부드럽게 배경을 애니메이션 처리했습니다.
- **색상 팔레트**: 실제 나눔로또의 공 색상 규정(노랑, 파랑, 빨강, 회색, 녹색)을 그대로 구현했습니다.

## 🎯 번호 추천 알고리즘과 수학적 개념
앱의 핵심은 과거 데이터를 단순히 보여주는 것을 넘어, 실질적인 알고리즘(Algorithm)에 맞춰 6개의 당첨 번호를 생성해 주는 것입니다. 

총 3가지의 추천 방식을 지원합니다:

### 1. 🔮 황금 밸런스 추천 (기본 모드)
역대 가장 많이 뽑힌 번호와 가장 적게 뽑힌 번호, 그리고 중간 빈도 번호를 2~3개씩 적절히 융합하여 6개의 공을 구성합니다. 
> *"뜨거운 번호와 차가운 번호의 균형 있는 선택"*을 지향하는 통계 중심 혼합 로직입니다.

### 2. 🔥 확률 집중 추천 
과거 출현 빈도가 높은 수일수록 뽑일 확률에 수식적 가중치(Weight)를 부여하여 뽑는 방식입니다. 예를 들어 역대 가장 많이 나온 숫자 34는 가중치가 높아 다른 공보다 상대적으로 더 자주 등장합니다.

### 3. 🎲 완전 무작위 (순수 확률)
과거의 확률 통계를 100% 무시하고 철저하게 무작위 난수(1/45)만을 생성하여 공정한 6개의 숫자를 제공합니다. 직관에만 100% 의존하고 싶을 때 적합합니다.

## 🛠️ 앱 사용 방법 (Getting Started)

### 사전 준비사항
1. 개발 환경으로 **Node.js 및 npm** 설치가 필요합니다.
2. 기존에 제공된 `로또 회차별 당첨번호_*.xlsx` 엑셀 데이터가 있다면 Python 스크립트(`generate_lotto_json.py`)를 통해 데이터 JSON 포맷으로 쉽게 업데이트할 수 있습니다. 이미 1212회차까지의 데이터를 가공해두었습니다.

### 서버 실행 (로컬 환경)
터미널을 열고 다음 명령어들을 순차적으로 실행하세요:
```bash
# 1. 앱 폴더로 이동합니다.
cd "lotto-app"

# 2. 패키지 설치를 완료합니다.
npm install

# 3. 개발 서버를 구동합니다.
npm run dev
```
이제 웹 브라우저(`http://localhost:5173`) 환경에서 화려하게 동작하는 로또 생성기를 만날 수 있습니다.

### 앱 화면 사용법 가이드
1. 앱 화면 중앙에 위치한 **[추천 방식 선택]** 드롭다운 상자에서 `황금 밸런스`, `확률 집중`, `완전 무작위` 중 원하는 로직을 클릭합니다.
2. 아래쪽의 파란색 **[번호 추첨하기]** 버튼을 누릅니다.
3. 잠시 번호를 추첨하는 시스템 렌더링 글자가 보인 뒤에 반짝이는 애니메이션과 함께 당첨 번호 6개가 등장합니다!

> *Tip: 화면 맨 하단에는 참고용으로 가장 마지막에 입력된 추첨 회차(1212회)의 실제 당첨 번호가 표시되어 있어 쉽게 번호를 비교해 보실 수 있습니다.*
